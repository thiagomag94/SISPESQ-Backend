API do SPIA v.1.0


Documentação

Esta é a documentação da API do SPIA, que permite a consulta de dados de forma rápida e eficiente.

Para mais informações, consulte o repositório no GitHub.

Endpoints

A API do SPIA (Sistema de Informações de Pesquisa e Análise) é a interface de backend central para a gestão e consulta de dados acadêmicos da Universidade Federal de Pernambuco (UFPE). Seu funcionamento principal envolve a coleta, conversão e armazenamento de dados de duas fontes primárias: Arquivos CSV: Contendo a relação de docentes da universidade e seus dados cadastrais. Currículos Lattes (XML): Extraindo informações detalhadas sobre a produção acadêmica de cada pesquisador. Todos os dados são processados e consolidados em um banco de dados MongoDB. A partir dessa base de dados consolidada, a API expõe uma série de rotas para consultar informações agregadas sobre a produção dos docentes, incluindo produção bibliográfica, técnica, artística, entre outras. O objetivo final é fornecer uma ferramenta poderosa para a geração de relatórios de pesquisa e para a análise estatística, permitindo a aplicação de filtros específicos em cada tipo de produção. Esta documentação é destinada principalmente aos desenvolvedores frontend do sistema SPIA e também a gestores que necessitem realizar consultas diretas aos dados.
